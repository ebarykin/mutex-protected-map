# Mutex-Protected Map (Go Test Task)

Минимальное тестовое задание на Go: потокобезопасная `map` с двумя счетчиками и конкурентным тестом.

## Содержимое

- `safe_map.go` — реализация `SafeMap` на `sync.Mutex`.
- `safe_map_test.go` — тест с 4 горутинами и проверкой инвариантов.
- `task.md` — исходное описание задания.

## Запуск

```bash
go test ./...
```

## Что проверяет тест

- для каждого ключа в диапазоне `1..year` итоговое значение равно `3`;
- `accessCount == year * 3`;
- `insertCount == year`.
